# ADR-004: Governance-First Architecture

## Status
Accepted

## Context
Need to prevent duplication and drift.

## Decision
All logic originates from governance system in Git.

## Rationale
Ensures consistency and central control.

## Alternatives Considered
- Embed logic in n8n
- Embed logic in code

## Consequences
### Positive
- Single source of truth

### Negative
- Requires strong governance discipline
